## About The Project

This project is a **Flask-based Issue and Vulnerability Tracking System** designed to help teams efficiently log, track, manage, and resolve issues and security vulnerabilities. It features a responsive dashboard, CRUD operations, search & filter capabilities, and detailed issue views.

## ✨ Features

* Add, View, Edit, and Delete issues
* Severity classification (Critical, High, Medium, Low)
* Status tracking (Open, In Progress, Resolved, Closed)
* Advanced search and filtering
* Dashboard with real-time statistics
* Responsive and modern UI
* SQLite database for lightweight deployment

## 🛠 Technologies Used

* **Backend**: Python, Flask
* **Database**: SQLite3
* **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
* **Templating**: Jinja2
* **Styling**: Custom CSS with modern design

## 🚀 Getting Started

### Prerequisites

* Python 3.8 or higher
* pip (Python package manager)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Khushig14/tracking-system.git
   cd tracking-system
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**
   ```bash
   python app.py
   ```

4. Deployment: Runs locally
